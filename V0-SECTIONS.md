## THEME

### Colors (JSM Dark Style)
- **Background Gradient:** `linear-gradient(90deg, rgb(4,7,29) 0%, rgb(12,14,35) 100%)`
- **Text Primary:** `#FFFFFF`
- **Text Secondary:** `#C1C2D3`
- **Text Muted:** `#BEC1DD`
- **Accent Purple:** `#CBACF9`
- **Accent Blue:** `#E4ECFF`
- **Card Background:** `rgba(17, 25, 40, 0.75)`
- **Border:** `rgba(255, 255, 255, 0.125)`

### Animations
- **spotlight:** Moving radial gradient effect
- **moveVertical:** Floating up/down
- **moveInCircle:** Rotating in circle
- **moveHorizontal:** Floating left/right
- **shimmer:** Background shimmer
- **Framer Motion:** Fade-in on scroll, staggered children

### Grid Background
- SVG grid pattern overlay for depth
- bg-grid-small class for subtle pattern
